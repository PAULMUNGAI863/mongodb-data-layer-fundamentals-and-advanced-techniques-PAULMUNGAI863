# 📚 PLP Bookstore - MongoDB Project

## 🚀 Overview
This project demonstrates MongoDB fundamentals — CRUD operations, advanced queries, aggregation pipelines, and indexing optimization.

## 🛠️ Setup Instructions

### 1. Install MongoDB
- Option 1: Install **MongoDB Community Edition** locally  
- Option 2: Sign up for **MongoDB Atlas** → [https://www.mongodb.com/atlas](https://www.mongodb.com/atlas)
